🛠 Step 1：註冊 GitHub 帳號

🧰 Step 2：建立一個新的 Repository

  1. 登入 GitHub。

🌐 Step 3：準備你要放的網站內容

✅ 方法一：自己從零寫 HTML/CSS（簡單頁面）

  </style>
</head>
<body>

</body>
</html>

✅ 方法二：Fork 別人的網站改（像你給的那個）

  1. 進到 https://github.com/xin-chun-1122/xin-chun-1122.github.io。

  2. 點右上角 Fork。

  3. 把它 fork 到你的帳號。

  4. 然後你可以在你的 repo 中修改 index.html、圖片、CSS 等，變成自己的風格。

🧑‍💻 Step 4：上傳你的網站檔案

  1. 在你的 repo 裡，點「Add file」➜「Upload files」。

  2. 把 index.html 和其他資源（圖片、CSS）上傳。

  3. 點擊「Commit changes」。

🚀 Step 5：開啟 GitHub Pages

通常如果你的 repo 名稱是 你的帳號.github.io，GitHub Pages 會自動啟用，只要你的 repo 有 index.html。

確認是否啟用：

  1. 打開 repo，點右上角「⚙️ Settings」。

  2. 左側點「Pages」或「Pages and Deployment」。

  3. Source 要選 main branch（或 master）。

  4. Save 後它會給你一個網址，通常就是 https://你的帳號.github.io/。
